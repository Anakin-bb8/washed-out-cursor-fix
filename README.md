# Washed out cursor fix
After a GPU change, monitor change, or similar, maybe your cursor looks washed out. This program fixes that restoring the colors of your cursors back to normal.

## What does this program do?
This program accesses the registry editor to change the "MouseTrails" value to -1. This for some reason fixes the cursor's colors.
The source code is written in Python and you can check what the software does.
